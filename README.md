✊✋✌️ Rock Paper Scissors

Este é um **jogo clássico de Pedra, Papel e Tesoura**, onde o jogador enfrenta o computador.

O projeto funciona diretamente no navegador e foi desenvolvido utilizando **JavaScript para controlar toda a lógica do jogo**.

⚠️ **Observação:**
Todo o projeto foi **desenvolvido de forma independente por mim**, incluindo a **estrutura em HTML, estilização em CSS e a lógica do jogo em JavaScript**, como prática de programação e aprendizado em desenvolvimento web.

# 🚀 Como funciona

O jogador escolhe uma das três opções disponíveis:

* ✊ **Pedra**
* ✋ **Papel**
* ✌️ **Tesoura**

Após a escolha do jogador, o computador gera **uma jogada aleatória** entre as mesmas opções.

O sistema então **compara as jogadas** e exibe o resultado da rodada.

---

# 🧠 Funcionalidades implementadas em JavaScript

* Captura da escolha do jogador através dos **botões**
* Geração de jogada **aleatória do computador**
* Comparação entre as jogadas
* Determinação automática do **resultado da partida**
* Atualização dinâmica das informações na interface
* Manipulação do **DOM** para exibir os resultados
